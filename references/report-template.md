# Markdown Daily Report Template

默认使用这套中文结构；除非用户明确要求，否则不要输出英文版式。

```md
# AI 日报 | YYYY-MM-DD

> 覆盖范围：YYYY-MM-DD 至 YYYY-MM-DD
> 生成时间：YYYY-MM-DD HH:MM TZ

## 今日要点

一句话概括今天最重要的 AI 变化。
```

## Writing Rules

- Keep the full report under roughly 900 Chinese characters unless the user asks for a longer brief.
- Prefer short bullets over dense prose.
- If there are fewer than three meaningful updates, say so explicitly and shorten the report.
- Do not add sections that are empty. Remove them cleanly instead.
